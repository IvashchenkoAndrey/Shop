##Тестовое задание

Что нужно сделать
1. Создать новую конфигурацию "Магазин".
2. Создать справочник товаров с теми реквизитами, которые Вы посчитаете нужными.
3. Создать регистраторы для фиксации поступления и реализации товаров.
4. Предусмотреть печатные формы для созданных регистраторов.
5. Создать отчет на СКД по проданным товарам за указанный период.
6. Создать отчет на СКД по остаткам товаров.
7*. Создать обработку, которая будет загружать курс доллара в конфигурацию с любого доступного в интернете сервиса по заданному расписанию.
8*. Добавить в отчет из пункта 5 суммы в валюте.
